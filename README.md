# rocket.watch

Website for watching rocket launches, otherwise - mom's spagetti.

![showcase](https://i.imgur.com/qJ6fE74.png)


## Prerequisites
- node.js
- npm (included with node)
- GNU Make >= 4.1

## Development workflow
In either `backend` or `frontend`
```
make help
```
